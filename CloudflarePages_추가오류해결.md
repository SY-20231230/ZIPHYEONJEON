# 🚨 추가된 빌드 에러들 모두 해결오나료!

아까 첫 번째 에러(`Registration.js` 앵커 에러)를 해결했지만, 에러 로그 전체를 올려주신 덕분에 숨어있던 **다른 파일들의 에러까지 전부 확인했습니다!**

## 1. 어떤 에러들이 숨어있었나요?
전체 로그를 살펴보니 Cloudflare 서버가 팅겨낸 깐깐한 이유들이 더 있었습니다.

*   **1) 사용하지 않는 변수 에러 (`no-unused-vars`)**
    *   `Loan.js`, `LoanDetail.js`, `RiskReport.js` 등에서 선언해놓고 막상 코드에서 쓰지 않은 찌꺼기 변수들을 ESLint가 허락하지 않았습니다.
*   **2) 또 다른 빈 링크 에러 (`jsx-a11y/anchor-is-valid`)**
    *   `RiskAnalysis.js`와 `Login.js`에도 아까처럼 주소가 비워진(href="#") `<a href="#">` 링크들이 남아 있었습니다.
